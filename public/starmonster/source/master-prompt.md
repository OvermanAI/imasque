# STAR MONSTER Character Master Prompt

## Core Identity

**Star Monster** is a dream creature that exists between Little Angel and Little Devil.

A fluffy cloud-shaped five-pointed star monster.

Cute, mysterious, and impossible to read.

Always looks harmless, but secretly possesses magical power that awakens only when injustice appears.

Japanese professional hand-drawn illustration style, premium mascot IP quality.

---

# Shape

### Cloud Star™

Body shape is a mix of a cloud and a five-pointed star.

Features:

- soft and fluffy
- chubby proportions
- rounded cloud-like edges
- five-pointed star silhouette
- no sharp corners
- symmetrical body
- front-facing
- plush toy silhouette
- looks like a creature born from dreams

The body should feel more like a soft cloud than a geometric star.

# Hidden Devil Tail

Usually invisible.

Only appears when Star Monster senses:

- bullying
- lies
- unfairness
- hurting the weak
- malicious destruction

Characteristics:

- very small
- candy-red color

- magical aura
- playful
- cute rather than evil

The tail must remain smaller than the wings.

Never oversized.
