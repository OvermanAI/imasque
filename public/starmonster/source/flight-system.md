

# StarMonster 飛行設定

## Wing Magic Flight System

StarMonster 會飛。

但牠不是靠翅膀拍動飛行。

牠的翅膀太小，不能像鳥一樣振翅，也不能像大型天使一樣展翅飛翔。

StarMonster 的飛行來自翅膀裡的夢境魔法。

當牠想要移動、表演、陪伴某個人，或前往一個正在閃爍的夢想時，牠的小小天使翅膀會發出柔和的月光與星光。

接著，StarMonster 會像雲朵一樣輕輕漂浮起來。

牠的飛行方式不是快速衝刺，而是柔軟、彈跳、漂浮、旋轉。

像一顆從夢裡飄起來的星星。

像一朵被月光托住的雲。

---

## 飛行規則

### 1. 不靠翅膀揮動

StarMonster 飛行時，翅膀不需要大幅拍動。

翅膀只會：

- 微微發光
    
- 小幅顫動
    
- 輕輕張開
    
- 散出星光粉塵
    
- 產生月光般的魔法浮力
    

### 2. 翅膀是魔法核心

StarMonster 的小天使翅膀不是飛行工具，而是夢境魔法的出口。

翅膀代表：

- 溫柔
    
- 守護
    
- 陪伴
    
- 希望
    
- 夢想浮力
    

### 3. 飛行姿態

StarMonster 飛行時，身體保持圓潤蓬鬆。

不可拉長。

不可出現手腳。

不可變成鳥類姿勢。

不可像超人一樣直線衝刺。

標準飛行感應該是：

- 漂浮
    
- 彈跳
    
- 滑行
    
- 旋轉
    
- 緩緩上升
    
- 像氣球一樣被夢托起來
    

### 4. 飛行速度

StarMonster 的飛行速度依情境而定。

**平常狀態**  
慢慢漂浮，像雲。

**表演狀態**  
輕快彈跳，像在星星之間跳舞。

**夢想勇氣模式**  
穩定向前，像被一道溫柔的光推著前進。

**感應不公平時**  
速度可以變快，但仍不可像戰鬥角色。

**魔法尾巴甦醒時**  
尾巴與翅膀一起發光，形成短暫的夢境加速，但仍保持可愛與柔軟。

## 視覺表現

### 平常飛行

- 翅膀微光
    
- 身體下方有小雲霧
    
- 周圍有淡淡星光
    
- 沒有尾巴
    

### 表演飛行

- 星光軌跡
    
- 身體微微旋轉
    
- 翅膀發亮
    
- 像在夢境舞台上移動
    

### 守護飛行

- 翅膀光芒稍強
    
- 身體向前傾
    
- 星光集中在前方
    
- 尾巴可若隱若現
    

### 魔法尾巴飛行

- 小小紅尾巴出現
    
- 尾巴產生糖果紅光點
    
- 翅膀散發月光
    
- 身後有粉紅與星黃色魔法軌跡
    

---

## 禁止事項

- 不可畫成大力揮動翅膀
    
- 不可把翅膀畫大
    
- 不可讓翅膀變成鳥翼
    
- 不可讓翅膀變成主視覺焦點
    
- 不可出現速度線過強的戰鬥感
    
- 不可像超級英雄飛行
    
- 不可像惡魔飛行
    
- 不可變成機械飛行
    
- 不可出現噴射效果
    
- 不可出現火焰推進
    
- 不可讓尾巴取代翅膀成為飛行主動力
